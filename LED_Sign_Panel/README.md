# 8x32 x4 LED Matrix Sign

## Pin mappings

Arduino Pin -> Ribbon Cable
* 4 -> CS1
* 5 -> CS2
* 6 -> CS3
* 7 -> CS4
* 10 -> WR
* 11 -> DATA
